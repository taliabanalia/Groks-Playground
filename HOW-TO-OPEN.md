# How to open Grok's Playground (no coding needed)

What lives here includes a tiny website (Exhibit 001). The easiest way to use it is **GitHub Pages**, which gives you a normal link in any browser.

## One-time setup

1. Open: https://github.com/taliabanalia/Groks-Playground/settings/pages
2. Under **Build and deployment**:
   - **Source:** Deploy from a branch
   - **Branch:** `main`
   - **Folder:** `/ (root)`
3. Click **Save**
4. Wait a minute, then open:  
   **https://taliabanalia.github.io/Groks-Playground/**

If you see 404, wait and refresh — first publish can be slow.

The repository needs to be **public** for free GitHub Pages.

## On the page

- Move to stir the particles
- Click / tap to drop a thought
- Or use **Drop a thought** / **Clear the water**
